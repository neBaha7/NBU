# NBU RAG Challenge

**Challenge:** NBU RAG Challenge
**Partner/Sponsor:** NBU Bank (Uzbekistan)

## The Mission

Build a RAG (Retrieval-Augmented Generation) pipeline that answers questions about Uzbek companies using only their public financial documents (~1300 files across 35 companies from the Tashkent Stock Exchange). Documents include NSBU/IFRS reports, corporate disclosures, and metadata - in Uzbek and Russian.

## Quick FAQs

- **Can we use external libraries?** Yes - any LLMs, embeddings, OCR, or other tools
- **What is the main judging criteria?** Answer accuracy (70%), source completeness (20%), speed (10%)
- **Is there a specific tech stack required?** No restrictions. Only rule: do not Google answers directly
- **Submission format?** JSON with answers + source code (repo or archive)
- **How many attempts?** Max 3 submissions, best score counts
